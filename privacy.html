Privacy Policy

Last updated: 2026-04-13

TikiTaka is built around a single idea: your snippets, keystrokes, and shortcuts belong on your device. This policy explains what little we touch and why.

Data we collect

License verification

When you activate a premium license, TikiTaka sends a hashed machine identifier (SHA-256 fingerprint) and your license key to the LemonSqueezy API for validation. No personally identifiable information is included in this request.

Update checks

TikiTaka periodically checks the GitHub Releases API to determine if a newer version is available. Only the current app version is sent; no personal data is transmitted.

Payment information

All payments are processed entirely by LemonSqueezy. TikiTaka never collects, stores, or has access to your credit card, billing address, or other payment details.

Cloud sync (optional)

TikiTaka offers an opt-in cloud sync feature. When enabled, an encrypted ZIP archive of your snippets, groups, and images is uploaded to the storage provider you choose:

  • Google Drive — saved into the app-specific hidden folder (drive.appdata scope). TikiTaka cannot read, modify, or delete any other files in your Drive.
  • WebDAV — saved directly to the WebDAV server you provide (Nextcloud, Synology, ownCloud, etc.). TikiTaka does not relay this data through any TikiTaka-controlled server.

Authentication tokens (OAuth refresh tokens, WebDAV credentials) are stored encrypted in your operating system keychain (macOS Keychain / Windows DPAPI) and never written to plain-text configuration files.

Data we do NOT collect

  • No analytics or telemetry — we do not track usage patterns, feature usage, or session data.
  • No keystroke logging — keyboard input is processed locally in real time for keyword matching and is never recorded, stored, or transmitted.
  • No snippet content collection — TikiTaka does not operate any server that sees your snippet data. Even with cloud sync enabled, your data goes directly to your chosen provider.

Data storage

All user data (snippets, settings, images) is stored locally in the application data directory on your device. If you opt into cloud sync, the same data is additionally synced to your chosen provider only.

Google API services disclosure

TikiTaka's use of Google API services adheres to the Google API Services User Data Policy, including the Limited Use requirements.

Specifically, TikiTaka only requests the https://www.googleapis.com/auth/drive.appdata scope, which restricts access to a hidden app-specific folder. TikiTaka cannot read, modify, or delete any other files in your Google Drive. Data accessed through Google APIs is used solely to provide the cloud sync feature you have enabled. It is not shared with third parties, used for advertising, or used to train AI/ML models.

개인정보처리방침

최종 수정: 2026-04-13

TikiTaka 는 한 가지 원칙 위에 만들어졌습니다 — 단축어, 키 입력, 데이터는 사용자의 기기에 머물러야 한다. 본 방침은 TikiTaka 가 다루는 최소한의 데이터가 무엇이며 그 이유가 무엇인지 설명합니다.

수집하는 정보

라이선스 인증

프리미엄 라이선스 활성화 시, 기기 식별을 위한 해시 값(SHA-256 핑거프린트)과 라이선스 키를 LemonSqueezy API 로 전송합니다. 개인 식별 정보는 포함되지 않습니다.

업데이트 확인

TikiTaka 는 GitHub Releases API 를 주기적으로 호출하여 새 버전 여부를 확인합니다. 현재 앱 버전만 전송되며, 개인 정보는 전송되지 않습니다.

결제 정보

모든 결제는 LemonSqueezy 를 통해 처리됩니다. TikiTaka 는 신용카드, 청구 주소 등 결제 정보를 수집·저장·접근하지 않습니다.

클라우드 동기화 (선택)

TikiTaka 는 선택적인 클라우드 동기화 기능을 제공합니다. 활성화하면 단축어, 그룹, 이미지가 담긴 ZIP 아카이브를 사용자가 선택한 저장소에 업로드합니다:

  • Google Drive — drive.appdata scope 의 앱 전용 숨김 폴더에 저장됩니다. TikiTaka 는 사용자의 다른 Drive 파일에 접근할 수 없습니다.
  • WebDAV — 사용자가 직접 지정한 WebDAV 서버 (Nextcloud, Synology, ownCloud 등)에 직접 저장됩니다. TikiTaka 가 운영하는 중간 서버는 존재하지 않습니다.

OAuth refresh token 과 WebDAV 자격증명은 운영체제 키체인 (macOS Keychain / Windows DPAPI)으로 암호화되어 저장되며, 평문 설정 파일에는 절대 기록되지 않습니다.

수집하지 않는 정보

  • 분석/텔레메트리 없음 — 사용 패턴, 기능 사용 빈도, 세션 데이터를 추적하지 않습니다.
  • 키 입력 기록 없음 — 키보드 입력은 키워드 매칭을 위해 로컬에서 실시간 처리되며, 기록·저장·전송되지 않습니다.
  • 단축어 내용 수집 없음 — TikiTaka 는 단축어 데이터를 받는 자체 서버를 운영하지 않습니다. 클라우드 동기화를 켜더라도 데이터는 사용자가 선택한 저장소로 직접 전송됩니다.

데이터 저장

모든 사용자 데이터(단축어, 설정, 이미지)는 사용자 기기의 앱 데이터 디렉토리에 로컬 저장됩니다. 클라우드 동기화를 활성화한 경우, 같은 데이터가 사용자가 선택한 저장소에만 추가로 동기화됩니다.

Google API 서비스 공개 사항

TikiTaka 의 Google API 사용은 Google API Services User Data Policy (Limited Use 요건 포함)를 준수합니다.

구체적으로 TikiTaka 는 https://www.googleapis.com/auth/drive.appdata scope 만 요청하며, 이는 앱 전용 숨김 폴더로 접근을 제한합니다. TikiTaka 는 사용자의 다른 Google Drive 파일에 대해 읽기·수정·삭제 권한이 없습니다. Google API 를 통해 접근하는 데이터는 사용자가 활성화한 클라우드 동기화 기능 제공 목적으로만 사용되며, 제3자와 공유되거나 광고에 사용되거나 AI/ML 모델 학습에 사용되지 않습니다.
